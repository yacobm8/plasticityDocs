Slide Curve CVs
Command
Slide
Shortcut
Shift + G
Button
-
Slide control vertices (CVs) along the tangential direction of the control cage defined by the spline curve.
Basic Usage
Select a CV or CVs you want to slide.
Press
Shift + G
to activate the Slide command.
Set the options in the Command Dialog and Keyboard Shortcuts.
Command Dialog
Distance:
Specify the distance of moved CVs.
Keyboard Shortcuts
Ctrl + Z
Undo:
Revert the last command execution, restoring the previous state.
U
Positive U:
Slide in a positive U direction.
Shift + U
Negative U:
Slide in a negative U direction.
N
Normal:
Slide in a normal direction.
Gizmo handles appear with specific direction controls. Use them to move CVs and adjust their positions precisely.
Confirm with OK in the Command Dialog or right-click.
