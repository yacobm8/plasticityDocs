Create a Solid from selected Faces and then manipulate its position.
Basic Usage
Select the Face of the base on which you want to create a Solid.
Press
Alt + D
to excute the command.
The Solid is created. Use the displayed Move Gizmo to move.
Set the options in the Command Dialog and the Keyboard Shortcuts.
Command Dialog

Specify the amount of movement of the X, Y, and Z axes.
