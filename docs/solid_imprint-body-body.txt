In Suggested Commands Menu
Create Edges on the target Solid or Sheet where it intersects with a tool Solid or Sheet. Optional Edges can also be created on the tool.
Basic Usage
Select the first Solid or Sheet that you want to use as the target for the intersection.
Select the second Solid or Sheet that you want to use as the tool for the intersection.
Set the options in the Command Dialog.
Command Dialog
Imprint tool:
Turn on
Imprint
to specify the
Complete tool
option.
Complete target:
None:
Does not extend the projected Edges on the target body.
Edge:
Extends the projected Edges until they intersect with other Edges on the target body.
Complete tool:
None:
Does not extend the projected Edges on the tool body.
Edge:
Extends the projected Edges until they intersect with other Edges on the tool body.
Confirm with OK in the Command Dialog or right-click.
