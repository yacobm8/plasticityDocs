Duplicate objects in a circular pattern by specifying the angle and number. Radial duplication is also possible.
Basic Usage
Select the object to duplicate.
Click the button
on the Right of Command Bar.
Click on the point you want to set as the center.
The duplication preview will be displayed.
Set the options in the Command Dialog and Keyboard Shortcuts.
Command Dialog
Mode:
Total:
Specify the total distance at which all duplications will be evenly arranged at the given degree value.
Spacing:
Specify the spacing distance for each of the duplications.
Degrees:
Specify the angle for duplication.
Number:
Specify the number of duplications.
Repeat:
Specify the number of repetitions in a radial pattern.
Repeat distance:
Specify the distance between each repetition in a radial pattern.
Keyboard Shortcuts
Shift + Wheel up
Add copy:
Increases the number of duplications.
Shift + Wheel down
Subtract copy:
Reduces the number of duplications.
Confirm with OK in the Command Dialog or right-click.
