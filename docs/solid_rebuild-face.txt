Basic Usage
Select a face or faces you want to rebuild.
Use the
Rebuild
command in the Command Palette or click on the Rebuild Button
on the Right of Command Bar.
Set the options in the Command Dialog.

Shrink projecting rows and columns of CVs to fit more closely to the boundary edges of the trimmed surface.
Tolerance:
Tolerance:
Specify the tolerance of surface CV count. Lower value results in large number of surface CVs and higher value results in small number of CVs. This determines the surface output quality.
