Trim Sheet with Patch
You can trim a sheet with the patch tool.
Select the target sheet you want to trim and temporarily hide it.
Select the base sheet and make an edge loop selection where there is a hole.
Enter in body selection mode, unhide the target sheet and make an additive selection to it.
Click the button
on the Right of Command Bar.
