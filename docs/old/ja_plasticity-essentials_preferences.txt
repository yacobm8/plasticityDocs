Plasticityの基本
設定
設定
設定ウィンドウを開くには、左上の P メニューをクリックし、
設定
を選択します。
設定を変更した後は、Plasticity を再起動する必要があります。
ナビゲーション
ナビゲーション設定
カーソルにズーム
:
有効にすると、ズームがカーソル位置に向かって行われます。（デフォルト:
有効
）
カーソルを中心に回転
:
有効にすると、ビューがカーソル位置を中心に回転します。（デフォルト:
有効
）
垂直ズーム
:
有効にすると、
Ctrl + 中ボタン
で上下にドラッグしてズームできます。（デフォルト:
有効
）
ズーム方向を反転
:
有効にすると、ズームの方向が逆になります。（デフォルト:
無効
）
ホイール方向を反転
:
有効にすると、マウスホイールのスクロール方向が逆になります。（デフォルト:
無効
）
ナビゲーションプリセット
他の 3D ソフトウェアと同じコントロールのプリセットを選択できます。
パフォーマンス
GPU のレンダリング品質と CPU の面品質を設定できます。
外観
アクセントカラー、サポートカラー、ビューポートの背景色、グリッドカラーを選択できます。
コントロールポイント、頂点、エッジ、カーブセグメントのサイズを制御するための追加オプションがあります。
バックアップ
緊急時に備えてバックアップを作成できます。
サーバー
Blender Bridge で使用するサーバーを設定します。
デフォルトの設定: ホスト: localhost, ポート: 8980。Blender アドオンの設定と一致させる必要があります。
SpaceMouse
3Dconnexion SpaceMouse の設定。
SpaceMouse Compact、SpaceMouse Pro、SpaceMouse Enterprise は Windows と macOS で動作確認済み。Linux での動作は未確認。
パン速度、回転速度、ズーム速度を調整できます。
一般
オフセット、押し出し、フィレットなどのコマンドを現在の選択に基づいて自動実行します。
有効にすると、オフセット、押し出し、フィレットなどのコマンドが自動的に実行されます。（デフォルト:
有効
）
詳細設定
Plasticity 起動時に前回の作業セッションを復元するかどうかを設定できます。（デフォルト:
有効
）
