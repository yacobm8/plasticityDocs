Im vorgeschlagenen Befehlsmenü
Erstellen Sie ein Rohr jeder Größe aus einer Curve oder einem Edge. Es kann für boolesche Operationen verwendet werden. Polygon- oder benutzerdefinierte Profiloptionen sind verfügbar.
Grundlegende Nutzung
Wählen Sie die Curve oder Edge aus, die Sie verrohren möchten. Wenn Sie mehrere Kurven oder Kanten auswählen, werden mehrere Pipes erstellt.
Drücken Sie
P
.
Ziehen Sie den gelben Punkt, um die Abschnittsgröße und das gelbe Quadrat, um die Rohrwandstärke zu bestimmen. Ziehen Sie den weißen Kreis, um den Drehwinkel festzulegen.
Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
Befehlsdialog
Select target bodies:
Gleiche Funktion wie der Befehl Boolean. Siehe
Boolean
Befehl für Details.
Select custom profile:
Wählen Sie Regions, Curves oder Faces aus, um die benutzerdefinierten Profile festzulegen.
Vertex count:
Geben Sie die Anzahl der Vertices für das Polygon des Profils an. Wenn Sie 0 angeben, wird es kreisförmig sein.
Section size:
Geben Sie die Abschnittsgröße an.
Thickness:
Geben Sie die Rohrwandstärke an.
Angle:
Geben Sie den Drehwinkel an.
Scale:
Geben Sie den Skalierungswert am Endpunkt des Rohrs an. Klicken Sie auf den Optionsdialog und geben Sie Ihren Wert manuell ein oder ziehen Sie mit der Maus von links nach rechts.

Hinweis
Der Befehl "Pipe" dient NICHT zum Erstellen maßstabsgetreuer Rohre.
Er dient zum schnellen Erstellen von cool aussehenden Rohren/Röhren.
Er führt viele Dinge wie Winkel als Polygone aus, die ihn für jede Art von Sanitär oder ähnlichem nicht maßstabsgetreu machen.
Um ein präzises Rohr zu erstellen, verwenden Sie den
Sweep
Befehl
