Neue Funktionen
Blender-Brücke
Die Blender-Brücke ermöglicht den schnellen Transfer von Mesh-Daten von Plasticity nach Blender. Dies wird als Blender-Add-On realisiert. Sobald Sie verbunden sind, können Sie einfach aktualisieren und Daten werden sehr schnell geladen. Sie müssen sich keine Sorgen machen, in eine temporäre Datei zu exportieren und alle Einstellungsoptionen richtig zu bekommen.
Selbst wenn Sie die Geometrie in Plasticity ändern, werden die in Blender festgelegten Materialien beibehalten. Zusätzlich wird die Gruppenstruktur in Plasticity als Sammlung in Blender wiedergegeben. Bei diesem Vorgang wird die Hierarchie von Blender mit dem Dokumentnamen von Plasticity als Bezeichner erstellt. Daher können mehrere Plasticity-Dokumente in ein einziges Blender-Dokument integriert werden.
Es gibt auch eine Live-Link-Funktion. Mit dem Live-Link können Sie Aktualisierungen in Echtzeit sehen. Das wirkliche Arbeitspferd der Blender-Brücke ist die Möglichkeit, Assets schnell zu importieren, sodass Sie sie mit Blender-Materialien, Blender-Beleuchtung und Raytracing sehen können. Sie können Ihre Objekte in einer Szene mit der echten Kamera anzeigen, die möglicherweise in einem Spiel oder in einem endgültigen Render verwendet wird, mit einem bestimmten FOV. Sie erhalten die volle Leistung von Blender, einschließlich dem Eltern von Dingen zu Empties, sodass Sie sie animieren, riggen und sehen können, ob Ihre mechanischen Teile zusammenpassen.
Der Arbeitsablauf ist ebenfalls ein fantastisches Erlebnis. Nachdem Sie das Plasticity-Dokument in die Blender-Szene eingefügt haben, können Sie immer noch die Facetten-Einstellungen ändern. Fast alle Facetten-Tools von Plasticity sind in Blender verfügbar. Das bedeutet, dass Sie schnell die für Ihr Modell benötigte Polygon-Dichte oder den Kantenfluss erhalten können, was zu einem schnelleren Arbeitsablauf führt.
Die Spezifikation von UV-Nähten und Schärfe in Blender wird in der aktuellen Version nicht beibehalten. Das Blender-Add-On bietet jedoch Funktionen, die Kanten automatisch markieren, die Geometrie automatisch entsprechend den Plasticity-Gesichtern oder Kanten auswählen und UV-Nähte zusammenführen. Diese Funktionen tragen ebenfalls zu einem schnelleren Arbeitsablauf bei.
Um die Blender-Brücke zu nutzen, müssen Sie das Add-On installieren und die Funktion aktivieren. Weitere Informationen finden Sie

Anpassbare radiale Menüs
Das radiale Menü ist ein Popup-Menü, das die schnelle Ausführung von Befehlen ermöglicht und mit einer Tastenkombination angezeigt werden kann. Standardmäßig werden zwei radiale Menüs bereitgestellt: das "Auswahlmodus-Radialmenü" zum Umschalten von Auswahlmodi und das "3D-Viewport-Einstellungs-Radialmenü" zum Ändern von 3D-Viewport-Einstellungen. Das radiale Menü kann von Benutzern angepasst oder neu erstellt werden und kann alle Befehle innerhalb von Plasticity integrieren. Die Installation ist erforderlich, um das standardmäßige radiale Menü zu verwenden.

Hierarchische Isolierung
Die Funktionalität des Isolieren-Befehls wurde erweitert, um das hierarchische Ausblenden von Objekten zu ermöglichen. Dies verbessert den Arbeitsablauf in komplexen Szenen erheblich. Nachdem Sie die angezeigten Objekte mit dem Isolieren-Befehl eingeschränkt haben, können Sie weitere Objekte innerhalb dieser Gruppe auswählen und den Isolieren-Befehl erneut ausführen, wobei nur die zuletzt ausgewählten Objekte sichtbar bleiben. Diese Funktion kann über mehrere Ebenen wiederholt werden. Der Isolierungsgrad wird auf dem Bildschirm angezeigt und durch Anklicken können Sie schrittweise zurückkehren, um mehr Objekte anzuzeigen.

Mehrere Fenster + Kopieren/Einfügen + Platzieren-Befehl für Kitbashing
Sie können jetzt mehrere Plasticity-Fenster öffnen, was die Arbeitseffizienz erhöht. Während Sie Objekte zwischen Fenstern kopieren und einfügen können, haben wir erweiterte Befehle eingeführt: 'Kopieren mit Platzierung' und 'Einfügen mit Platzierung'. Diese funktionieren ähnlich wie der Platzieren-Befehl, aber zwischen verschiedenen Fenstern. In einem Fenster können Sie ein Objekt mit einem Referenzpunkt kopieren und es dann mit diesem Punkt in einem anderen Fenster einfügen. Diese Funktion ist besonders praktisch für Kitbashing.

Viele Verbesserungen am Gliederung
Rechtsklicken Sie auf einen Gruppenordner, um auf verschiedene zugehörige Befehle zuzugreifen, wie z.B. "Aktuelle Auswahl zur Gruppe hinzufügen", "Alles auswählen", "Neue Untergruppe mit aktueller Auswahl", alle Gruppen reduzieren oder erweitern und Materialien ändern.
Scrollen zur Auswahl: Es wurde eine Verknüpfung bereitgestellt, um das in der 3D-Ansicht ausgewählte Objekt im Gliederung   zu lokalisieren.
Auswahl isolieren: Es wurde auch eine Verknüpfung bereitgestellt, um alle Gruppen und Abschnitte zusammenzufalten, außer demjenigen, der Ihr ausgewähltes Objekt enthält.
Sie können Massenzustandsänderungen durchführen - wie das Ausblenden, Deaktivieren, Sperren - indem Sie mehrere Objekte umbenennen oder Materialien ändern.
Alt-Klick auf den Gruppenordner, um alle Objekte in diesem Abschnitt auszuwählen.
Rechtsklick auf eine Abschnittsbezeichnung ("Solids", "Sheets", "Curves") und "Alles auswählen" auswählen, um alle Objekte in diesem Abschnitt auszuwählen.

Chordale Rundungen
Die chordale Option wurde dem soliden Rundungswerkzeug hinzugefügt. Das Hauptmerkmal einer chordalen Rundung besteht darin, dass sie eine konstante Länge für die Saite der Rundung beibehält, unabhängig von den abgerundeten Winkeln. Diese Gleichförmigkeit kann besonders vorteilhaft sein, wenn Sie eine visuelle Konsistenz über verschiedene Teile eines Modells anstreben.
