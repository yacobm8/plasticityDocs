Grundlegende Nutzung
Einen offenen hohlen Solid erstellen
Wählen Sie die Flächen des Solids aus, die Sie aushöhlen möchten.
Klicken Sie auf die Schaltfläche
Bewegen Sie den gelben Punkt, um die Dicke der Wand anzugeben.
Legen Sie die Optionen im Befehlsdialog fest:
Befehlsdialog
Dicke:
Geben Sie die Dicke der Wand an.
Lokal:
Deaktivieren Sie dies, wenn Sie einen Solid aushöhlen.
Tastenkombinationen
D
Geben Sie die Dicke durch Bewegen des Cursors an.
Bestätigen Sie mit OK im Befehlsdialog oder mit Rechtsklick.
Ein geöffneter hohler Solid wird erstellt.
