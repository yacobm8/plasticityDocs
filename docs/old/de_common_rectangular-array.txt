Schaltfläche
Im vorgeschlagenen Befehlsmenü
Duplizieren Sie Objekte in einem linearen Muster, indem Sie die Anzahl der Duplikate angeben. Optional können Sie eine andere Richtung definieren, um ein rechteckiges Array zu erstellen.
Grundlegende Nutzung
Wählen Sie das Objekt aus, das Sie duplizieren möchten.
Klicken Sie auf die Schaltfläche
rechts von der Befehlsleiste.
Die Duplikationsvorschau wird angezeigt.
X
/
Y
/
Z
X/Y/Z-axis:
Binden Sie an die X-, Y- oder Z-Achse
.
Klicken Sie auf die gewünschte Richtung.
Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
Modus:
Ausdehnung:
Geben Sie den Abstand der Ausdehnung an, bei dem alle Duplikate gleichmäßig angeordnet werden.
Abstand:
Geben Sie den Abstand für jedes Duplikat an.
Befehlsdialog
Abstand 1:
Geben Sie den Abstand an.
Anzahl:
Geben Sie die Anzahl der Duplikate an.
Tastenkürzel
Shift + Mausrad nach oben
Kopie hinzufügen: Erhöht die Anzahl der Duplikate.
Shift + Mausrad nach unten
Kopie reduzieren: Verringert die Anzahl der Duplikate.
Wenn Sie in eine andere Richtung duplizieren möchten, klicken Sie im Dialog auf
Select endpoint 2
.
Klicken Sie mit der rechten Maustaste auf die gewünschte Duplikationsrichtung.
Legen Sie die Optionen im Befehlsdialog fest.
Befehlsdialog
Distance 2:
Geben Sie den Abstand an.
Repeat:
Geben Sie die Anzahl der Wiederholungen an.
Tastenkombinationen
Umschalt + Mausrad hoch
Add copy:
Erhöht die Anzahl der Duplikate.
Umschalt + Mausrad runter
Subtract copy:
Verringert die Anzahl der Duplikate.
Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
