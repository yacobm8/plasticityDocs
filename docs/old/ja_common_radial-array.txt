Radial Array
Radial Array（円形状配列で複製）
コマンド
Radial Array
ショートカット
-
ボタン
コマンドバー右
角度と数を指定して、円状のパターンでオブジェクトを複製できます。放射状の複製も可能です。
基本的な使い方
複製したいオブジェクトを選択します。
コマンドバー右のボタン
をクリックします。
中心にしたい点をクリックします。
複製プレビューが表示されます。
コマンドダイアログとキーボードショートカットでオプションを設定します。
コマンドダイアログ
モード:
合計:
指定した次数値で、すべての複製が均等に配置される合計距離を指定します。
間隔:
各複製の間隔距離を指定します。
Degrees:
複製する角度を指定します。
Number:
複製する数を指定します。
Repeat:
放射状に繰り返す回数を指定します。
Repeat distance:
放射状に繰り返す間隔を指定します。
キーボードショートカット
Shift
+ ホイール上回転
Add copy:
複製の数を増やします。
Shift
+ ホイール下回転
Subtract copy:
複製の数を減らします。
コマンドダイアログのOKをクリックするか、右クリックで確定します。
