Im Menü Vorgeschlagene Befehle
Erstellen Sie eine neue Kurve, indem Sie eine Kurve auf ein Solid oder Sheet projizieren. Wählen Sie die Projektionsmethode normal oder Richtungsvektor aus. Bidirektional ist ebenfalls verfügbar.
Grundlegende Nutzung
Wählen Sie das Solid oder Sheet aus, auf das projiziert werden soll.
Drücken Sie
I
oder klicken Sie auf die Schaltfläche.
Wählen Sie die Kurve aus, die projiziert werden soll.
Legen Sie die Optionen im Befehlsdialog fest:
Befehlsdialog
Method:
Normal:
Projiziert in Richtung der positiven Z-Achse basierend auf der Konstruktionsebene.
Vector:
Projiziert in eine lineare Richtung.
Bidirectional
und
Hide occlusion
werden aktiviert.
Bidirectional:
Wenn aktiviert, erfolgt die Projektion auch in die negative Richtung der angegebenen Achse.
Hide occluded:
Wenn aktiviert, erfolgt die Projektion nicht auf versteckten Faces in Richtung der positiven Achse.
Rechtsklick oder Klick auf OK im Befehlsdialog zur Bestätigung.
Die projizierte Kurve wird erstellt.
