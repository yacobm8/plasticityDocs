ツールバー
連続性を考慮してふたつのカーブを繋げることができます。また、エッジ上、フェース上、構築平面上の任意の点も繋げることができます。
基本的な使い方
❗
注意
ポイントを選択する前に、
グリッドスナップ
と
オブジェクトスナップ
を有効にしてください。
ℹ️
始点と終点は、カーブ上、エッジ上、フェース上、または構築平面上の任意の点を選択することができます。

をクリックします。

終点をクリックします。
コマンドダイアログとキーボードショートカットでオプションを設定します。
コマンドダイアログ
T:
Value 1/2:
始点/終点のセグメント上の相対位置を指定します。
Direction:
Direction 1/2
: 始点/終点の方向を反転します。
Tension 1:
Value 1/2:
始点/終点の第1制御点(CV)から第3CVに影響するテンション値を指定します。
Tension 2:
Value 1/2:
始点/終点の第2CVと第3CVに影響するテンション値を指定します。
Tension 3:
Value 1/2:
始点/終点の第3CVに影響するテンション値を指定します。
Start curvature:
G0, G1, G2, G3:
始点での
曲線の連続性
を指定します。
End curvature:
G0, G1, G2, G3:
終点での
曲線の連続性
を指定します。
Trim:
オンにすると、ブリッジからはみ出たカーブはトリムされます。
コマンドダイアログでOKを押すか、右クリックで確定します。
