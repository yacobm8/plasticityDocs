Draft Face
Draft Face（フェイスを傾ける）
コマンド
Draft Face
ショートカット
-
ボタン
推奨コマンドメニューで
フェイスを傾けることができます。
基本的な使い方
リファレンス面となるフェイスを選択します。
推奨コマンドメニュー (
Suggested Commands
) 内の
ボタンをクリックしてください。
Ctrl
ボタンを押しながら、ドラフトする面を選択します。
白い円をドラッグしてドラフトの角度を指定するか、ギズモを使って参照平面の位置を調整します。
コマンドダイアログとキーボードショートカットでオプションを設定します。
コマンドダイアログ
Degrees:
傾ける角度を指定します。
Grow:
凹んだ内側のフェイスを傾けるとき:
Moving:
傾斜面が外壁に到達したら、外壁を延長します。
Fixed:
外壁で停止します。
None:
外壁の影響を受けず、傾斜面が進み続けます。
キーボードショートカット
A
Angle adjacent
カーソルを動かして角度を指定します。
コマンドダイアログのOKボタンをクリックするか、右クリックして確定します。

Note
非均一な面にドラフトコマンドを適用することは可能ですが、いくつかの制限があります。ドラフト角度を指定することはできますが、参照平面を移動することはできません。
