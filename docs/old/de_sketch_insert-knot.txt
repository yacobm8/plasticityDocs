Fügt einen Kontrollpunkt (Control-Vertex) in eine Spline-Kurve ein.
Grundlegende Verwendung
Klicken Sie auf die Schaltfläche
in der Werkzeugleiste.
Klicken Sie auf den Punkt, an dem Sie einen Kontrollpunkt einfügen möchten.
Ein Kontrollpunkt wird eingefügt, wodurch ein zusätzlicher Punkt zur Spline-Kurve hinzugefügt wird.

🗒️
Hinweis
Das Einfügen eines Kontrollpunktes erfordert, dass die Kurve eine Spline ist; andernfalls hat es bei einer normalen segmentierten Kurve keine Wirkung.
