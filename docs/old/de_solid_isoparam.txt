Befehl
Isoparam
Tastenkürzel
Strg + R
Schaltfläche
-
Kanten auf einer Face eines Solid oder Sheet in der U- oder V-Richtung erstellen. Mehrere Kanten können gleichzeitig erstellt werden.
Grundlegende Nutzung
Drücken Sie
Strg + R
.
Platzieren Sie den Mauszeiger an der Position, an der Sie die Isoparams erstellen möchten.
Legen Sie die Optionen in den Tastenkombinationen fest.
Tastenkombinationen
Tab
U/V direction:
Wechseln Sie die Richtung des zu erstellenden Isoparam.
Umschalt
+ Mausrad hoch
Add cut:
Erhöht die Anzahl der Isoparams.
Umschalt
+ Mausrad runter
Subtract cut:
Reduziert die Anzahl der Isoparams.
Bestätigen Sie mit einem Rechtsklick.
