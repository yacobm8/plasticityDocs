Cut Curve
Befehl
Cut Curve
Tastenkürzel
C
Schaltfläche
Im vorgeschlagenen Befehlsmenü
Schneiden Sie die Ziel-Curve mit einer Schneide-Curve oder Face. Wenn eine Curve der Schneide dient, wird der Schnitt basierend auf einer von dieser Curve erzeugten Oberfläche gemacht.
Grundlegende Nutzung
Drücken Sie
C
.
Drücken Sie erneut
C
, um zum
Cut Curve
Befehl zu wechseln.
Legen Sie das Ziel und den Schneide fest.
Befehlsdialog
Select target bodies
: Wählen Sie die Curve aus, die geschnitten werden soll.
Select cutters
: Wählen Sie die Curve oder Face aus, die als Schneide verwendet werden soll.
Tastenkombinationen
S
Screen space
: Legt die Schnittrichtung so fest, dass sie mit der sichtbaren Richtung übereinstimmt.
Bestätigen Sie mit OK im Befehlsdialog oder klicken Sie mit der rechten Maustaste.
