Einrichtung von Blender Bridge
Um das Feature von Blender Bridge zu nutzen, müssen Sie das Add-On in Blender installieren und die Funktion innerhalb von Plasticity aktivieren.
Blender Add-on installieren
Laden Sie das Blender Bridge Add-on aus dem offiziellen GitHub-Repository herunter. Hier ist der direkte Link zur neuesten Version:
plasticity-blender-addon-main.zip
(opens in a new tab)
Starten Sie Blender.
Installieren Sie plasticity-blender-addon-main.zip als Blender Add-on wie gewohnt.
Öffnen Sie Blender.
Navigieren Sie zu Bearbeiten > Einstellungen > Add-ons.
Klicken Sie auf "Installieren..." und wählen Sie die heruntergeladene Zip-Datei aus.
Bitte beachten Sie, dass es einige Zeit dauern kann, bis "[ ]Objekt: Plasticity" in der Liste erscheint.
Sobald es installiert ist, können Sie "Plasticity" in der Seitenleiste der 3D-Ansicht finden.
Funktion in Plasticity aktivieren
Starten Sie Plasticity.
Öffnen Sie Einstellungen aus dem P-Menü, das sich oben links auf dem Bildschirm befindet.
Öffnen Sie auf der linken Seite Server und setzen Sie ein Häkchen bei Aktiviert.
Jetzt sind Sie bereit, das Feature von Blender Bridge zu nutzen.
