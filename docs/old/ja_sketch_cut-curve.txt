Cut Curve
Cut Curve（カーブを切断）
コマンド
Cut Curve
ショートカット
C
ボタン
推奨コマンドメニューで
カーブを、カーブまたはフェイスで切断できます。
カーブで切断するときは、そのカーブから生成された曲面を基にしてカットが行われます。
基本的な使い方
キーボードショートカット
C
を押します。
もう一度
C
を押して、
Cut Curve
コマンドに切り替えます。
ターゲットとカッターを指定します。
コマンドダイアログ
Select target bodies
: カットするカーブを選択します。
Select cutters
: カッターとして使用するカーブまたはフェイスを選択します。
キーボードショートカット
S
Screen space:
カット方向を見た目の方向からにします。
コマンドダイアログのOKをクリックするか、右クリックで確定します。
