Paste With Placement
Shortcut

-
Paste objects with a reference point. Use it alongside the
Copy with Placement
command for similar functionality to the
Place
command. This is particularly useful for placing objects in different windows.
Basic Usage
Copy with Placement
Command
Select the objects you wish to copy.
Press
Ctrl + Shift + C
.
Click the reference point on the object you're copying.
To paste in a different window, switch to that target window.
Paste with Placement
Command
Press
Ctrl + Shift + V
.
Click the destination reference point in the target window.
