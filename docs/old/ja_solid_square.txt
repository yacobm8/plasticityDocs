Square は、UV の次数とスパンを明示的に制御できるパラメトリック機能を備えた、Xnurbs ベースのコマンドで、高品質な四角形の面を作成することができます。
基本的な使い方
使用する境界エッジを選択し、
Square コマンド
を実行します。
コマンドパレットで
Square
と入力します。
コマンドダイアログでオプションを設定します。
コマンドダイアログ
Continuity:
自由 (Free), G0, G1, G2
Free:
境界の連続性を無視し、サーフェスの品質を向上させます。主に後で修正するための初期サーフェス作成に適しています。
G0, G1, G2:
サーフェスの連続性
を指定します。
Flatness:
XNurbs と同様に、このオプションは生成されるサーフェスの平坦度と品質に影響します。値を高くすると高品質なサーフェスが生成され、低くすると品質が下がります。
Tension:
サーフェスの曲率遷移を調整します。
Degree:
UV の次数を指定します。
Spans:
UV のスパン値を指定します。
Boundary condition:
UV フローの条件を指定します。
Natural:
UV フローの制約なし。
Normal:
UV アイソカーブを互いに直交させます。
Next:
境界条件が比較的平坦な場合、UV を直交フローとして生成します。
Adjacent:
生成されるサーフェスの V アイソカーブを隣接するカーブに揃えます。
Weight:
曲線やエッジの入力制約の適用強度を制御します。
低い値: 制約を緩めて、サーフェス品質や反射を向上。
高い値: 制約を厳密に適用し、指定した
Degree
や
Span
の UV フローに正確に整列。
Scale:
境界エッジ上に表示される赤い線のサイズを調整します。
これは
「エッジ偏差プロット (Edge Deviation Plot)」
を表し、生成されたサーフェスの境界と入力サーフェスの距離を示します。
赤色は 0.01mm (1e-5m) 以上の偏差を示し、入力ジオメトリと一致しない領域を強調します。
コマンドダイアログで
OK
を押すか、右クリックして確定します。

Square でのロフト
代替として、Square コマンドはロフト手法として使用することができ、2 つの接触していないエッジを選択することで適用できます。この方法では、度数やスパン値、境界条件（必要に応じて）、平坦度、ウェイト影響などのパラメータを指定することができ、より柔軟な制御が可能です。一方、標準的なロフトを使用すると、制御オプションが限られた固定されたサーフェスが生成されます。

Square でのスイープ
Sweep コマンドと同様に、Square コマンドは同等の結果を生成できますが、より柔軟で詳細な制御が可能です。この結果は、2 つまたは 3 つの接触しているエッジを選択することで達成できます。

Square でのパッチング
Square コマンドは、Patch コマンドと同様に、四辺の境界エッジを持つ穴をサーフェスで埋めることができます。
