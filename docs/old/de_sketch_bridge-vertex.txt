Bridge

Verbindet zwei Vertices, die G0/G1/G2/G3-kontinuierlich sein können. Die Spannung ist einstellbar.
Grundlegende Nutzung
Wählen Sie zwei Vertices, die Sie verbinden möchten.
Verwenden Sie den
Bridge
Befehl im Command Palette.
Legen Sie die Optionen im Command Dialog und den Tastenkombinationen fest.
Befehlsdialog
Tension 1:
Value 1/2:
Der Spannungswert beeinflusst den 1. Control-Vertex(CV) bis zum 3. CV des Startpunktes/Endpunktes.
Tension 2:
Value 1/2:
Der Spannungswert beeinflusst den 2. CV und den 3. CV des Startpunktes/Endpunktes.
Tension 3:
Value 1/2:
Der Spannungswert beeinflusst den 3. CV des Startpunktes/Endpunktes.
Start curvature:
G0, G1, G2, G3:
Gibt die
Kontinuität der Curve
am Startpunkt an.
End curvature:
G0, G1, G2, G3:
Gibt die
Kontinuität der Curve
am Endpunkt an.
Bestätigen Sie mit OK im Command Dialog oder klicken Sie mit der rechten Maustaste.
