Verschieben Sie die ausgewählten Face, indem Sie den Abstand angeben. Eine optionale Angabe des benachbarten Winkels ist verfügbar.
Grundlegende Nutzung
Wählen Sie die Face aus, die Sie versetzen möchten.
Wenn eine Face ausgewählt ist, wird der Standardbefehl,
Push Face
, automatisch ausgeführt.
Bewegen Sie den gelben Punkt, um den Abstand anzugeben.
Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
Befehlsdialog
Distance:
Geben Sie den Abstand an.
Angle Adjacent:
Geben Sie den Winkel an.
Grow:
Wenn die konkave innere Face verschoben wird.
Moving:
Bewegen Sie die gesamte Wand, wenn die Verschiebung zur Außenwand gelangt.
Fixed:
Stoppen Sie an der Außenwand.
None:
Fortsetzen, ohne von der Außenwand beeinflusst zu werden oder diese zu beeinflussen.
Tastenkombinationen
D
Distance
Geben Sie den Abstand durch Bewegen des Cursors an.
A
Angle adjacent
Geben Sie den Winkel durch Bewegen des Cursors an.
Tab
Grow mode
Wechselt den Wachstumsmodus.
Bestätigen Sie mit OK im Befehlsdialog oder mit Rechtsklick.

Abhängiger Versatz
Passen Sie ausgewählte Flächen einer anderen an und versetzen Sie diese in einer einzigen Operation.
Wählen Sie die Flächen aus, die Sie versetzen möchten.
Wenn eine Fläche ausgewählt ist, wird der Standardbefehl
Push Face
automatisch ausgeführt.
Beginnen Sie damit, die Fläche(n) zu versetzen, und lassen Sie dann das Versetzungswerkzeug los.
Wählen Sie die Fläche aus, die Sie anpassen möchten. Falls erforderlich, verschieben Sie die Fläche, indem Sie das Versetzungswerkzeug ziehen und loslassen oder einen Versatzwert eingeben.
Bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.
