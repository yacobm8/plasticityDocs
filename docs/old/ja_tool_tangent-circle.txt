2つのカーブに接する円を作成できます。
基本的な使い方
ツールバーの
ボタンをクリックします。
円が接する最初と2番目の曲線をクリックし、必要に応じて3番目のエッジへの接線を見つけることができます。
カーソルを動かして円の大きさを調整します。
コマンドダイアログ
Select first curve segment
: ハイライトされているとき、クリックして最初の曲線セグメントを選択します。
Select second curve segment
: ハイライトされているとき、クリックしてふたつめの曲線セグメントを選択します。
Radius
: 円の半径を指定します。
コマンドダイアログのOKをクリックするか、右クリックで確定します。
円が作成されます。
