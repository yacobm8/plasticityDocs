Project Curve Body（カーブをボディに投影）
コマンド
Project
ショートカット
I
ボタン
提案コマンドメニュー内
カーブをソリッドまたはシートに投影して、カーブを作成できます。
基本的な使い方
投影したいソリッドまたはシートを選択します。
キーボードショートカット
I
またはボタンを押します。
投影するカーブを選択します。
コマンドダイアログのオプションを設定します。
コマンドダイアログ
Method:
Normal:
構築平面を基準に Z 軸正方向に投影します。
Vector:
直線方向に投影します。
Bidirectional
と
Hide occlusion
が有効になります。。
Bidirectional:
オンにすると指定した軸の負の方向にも投影されます。
Hide occluded:
オンにすると指定した軸の正の方向で、隠れているフェイスには投影されなくなります。
コマンドダイアログのOKをクリックするか、右クリックで確定します。
投影されたカーブが作成されます。
