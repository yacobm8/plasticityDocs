推奨コマンドメニューで
Offset カーブ コマンドは選択されたオブジェクトに基づいて実行されます：
Offset 頂点:
頂点が選択されている場合は、
Offset 頂点
を参照してください。
Offset 平面カーブ:
カーブが選択されている場合は、
Offset 平面カーブ
を参照してください。
Offset 領域:
領域が選択されている場合は、
Offset 領域
を参照してください。
Offset 面ループ:
面が選択されている場合は、ソリッドコマンドのセクションにある
Offset 面ループ
を参照してください。
Offset エッジ:
エッジが選択されている場合は、ソリッドコマンドのセクションにある
Offset エッジ
を参照してください。
