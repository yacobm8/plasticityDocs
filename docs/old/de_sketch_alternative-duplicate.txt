Alternative Duplicate
Befehl
Alternative Duplicate
Tastenkürzel
Alt + D
Schaltfläche
-
Projizieren Sie die Curves auf die aktuelle Bauebene entlang der normalen Richtung, wodurch eine Curves erzeugt wird.
Grundlegende Nutzung
Legen Sie die gewünschte Bauebene für die Projektion fest.
Wählen Sie die Curves aus, auf die projiziert werden soll.
Drücken Sie
Alt + D
, um den Befehl auszuführen.
Die projizierten Curves werden auf der aktuellen Bauebene erstellt.
Optional können Sie die Curves mit dem Move Gizmo verschieben.
Du kannst auch die Bewegungsrichtung einschränken.
Tastenkombinationen
Umschalt + X
/
Umschalt + Y
/
Umschalt + Z
X, Y, Z Ebene:
Beschränke auf die X, Y oder Z-Ebene.
G
Screen space:
Beschränkung auf die sichtbare Ebene.
F
Freestyle:
Beschränke die Richtung durch zwei Punkte.
V
Pivot:
Beschränke die Richtung durch Klicken auf die Ebene.
W
Worldspace:
Löse die Richtungsbeschränkung auf.
Klicken Sie mit der rechten Maustaste oder klicken Sie auf OK im Befehlsdialog, um zu bestätigen.
