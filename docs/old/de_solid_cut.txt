Im vorgeschlagenen Befehlsmenü
Schneiden Sie das Ziel-Solid oder Sheet mit einer Cutter-Curve oder Face. Wenn eine Curve der Cutter ist, wird der Schnitt basierend auf einer von dieser Curve erzeugten Oberfläche durchgeführt.
Grundlegende Nutzung
Drücken Sie
C
, um den Befehl auszuführen.
Spezifizieren Sie das Ziel und den Cutter.
Befehlsdialog
Select target bodies:
Wählen Sie das Solid oder Sheet aus, das geschnitten werden soll.
Um das Ziel zu ändern, klicken Sie auf
Select target bodies
und wenn es hervorgehoben ist, wählen Sie das Ziel.
Select cutters:
Wählen Sie die Curve oder Face aus, die als Cutter verwendet werden soll.
Um die Cutter zu ändern, klicken Sie auf
Select cutters
und wenn es hervorgehoben ist, wählen Sie die Cutter.
