Duplizieren Sie Objekte entlang einer Kurve, indem Sie die Anzahl der Duplikate angeben. Optionen zum Drehen, Skalieren und Ausrichten sind verfügbar.
Grundlegende Nutzung
Wählen Sie das Objekt aus, das Sie duplizieren möchten.
Verwenden Sie den
Curve Array
Befehl in der Befehlspalette.
Wählen Sie die Kurve aus, die als Pfad verwendet werden soll.
Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
Befehlsdialog
Number:
Geben Sie die Anzahl der Duplikationen an.
Twist:
Geben Sie den Verdrehungswinkel entlang der Kurve in Grad an.
Scale:
Geben Sie den Skalierungsfaktor entlang der Kurve an.
Alignment:
Parallel:
Duplizieren Sie die Kurven parallel, unabhängig vom Pfad.
Normal:
Duplizieren Sie die Kurven mit Winkeln, die an den Pfad ausgerichtet sind.
Tastenkombinationen
Umschalt + Rad hoch
Add copy:
Erhöht die Anzahl der Duplikationen.
Umschalt + Rad runter
Subtract copy:
Verringert die Anzahl der Duplikationen.
Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
