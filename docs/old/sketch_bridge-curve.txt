Bridges two Curves that can be G0/G1/G2/G3 continuous. Tension is adjustable. Auto trim is possible.

Command Dialog
T:
Value 1/2:
The relative position on the segment of the starting point/ending point.
Direction:
Direction 1/2:
Reverse the direction of the starting point/ending point.
Tension 1:
Value 1/2:
The tension value affected 1st Control-Vertex(CV) to 3rd CV of the starting point/ending point.
Tension 2:
Value 1/2:
The tension value affected 2nd CV and 3rd CV of the starting point/ending point.
Tension 3:
Value 1/2:
The tension value affected 3rd CV of the starting point/ending point.
Start curvature:
G0, G1, G2, G3:
Specify the
continuity of the  Curve
at the starting point.
End curvature:
G0, G1, G2, G3:
Specify the
continuity of the  Curve
at the ending point.
Trim:
When turned on, any Curve that extends beyond the bridge will be trimmed.
Confirm with OK in the Command Dialog or right-click.
