Patchen mit Leit-Curves
Sie können Löcher entlang von Edges mit Leit-Curves patchen.
Erstellen Sie die Leit-Curves.
Fügen Sie an jedem Punkt, an dem die Leit-Curve die Oberfläche berührt, einen Isoparam hinzu. Verwenden Sie den
Isoparam
-Befehl.
Wählen Sie die Edges aus, die das Loch umgeben.
Klicken Sie auf den Button
rechts von der Befehlsleiste.
Wählen Sie die Leit-Curves aus.
Legen Sie die Optionen im Befehlsdialog fest.
Die Optionen sind wie oben.
Bestätigen Sie mit OK im Befehlsdialog oder mit Rechtsklick.

Mehrere Löcher patchen
Sie können mehrere Löcher auf einmal patchen.
Wählen Sie die Sheets mit mehreren Löchern aus, die Sie patchen möchten.
Klicken Sie auf den Button
rechts von der Befehlsleiste.
Löcher werden gepatcht und zu Solid gemacht.

Schicht mit Patch trimmen
Sie können eine Fläche mit dem Patch-Werkzeug zuschneiden.
Wählen Sie die Ziel-Fläche aus, die Sie zuschneiden möchten, und blenden Sie sie vorübergehend aus.
Wählen Sie die Basis-Fläche aus und treffen Sie eine Kantenloop-Auswahl an der Stelle, wo sich ein Loch befindet.
Wechseln Sie in den Körperauswahlmodus, blenden Sie die Ziel-Fläche wieder ein und fügen Sie eine additive Auswahl hinzu.
Klicken Sie auf die Schaltfläche
rechts in der Befehlsleiste.
Stellen Sie die Optionen im Befehlsdialog ein.
Command Dialog

Trim to sheet:
Trimmen Sie die Ziel-Fläche, um sie mit dem Endbereich der Basis-Fläche auszurichten.
Tolerance:
Geben Sie die Toleranz für den Patch an.
