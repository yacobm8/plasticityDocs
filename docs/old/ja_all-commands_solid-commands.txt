コマンドリスト & ショートカットキー
ソリッドコマンド
ソリッドコマンド
コマンド名
ショートカット
機能
Boolean
Q
ブーリアンソリッドおよび/またはシート。
Cut
C
ソリッドとシートをカットします。
Solid and Sheet Commands
Alternative Duplicate
Alt+D
代替複製。
Constrained Surface
制約された表面を作成します。
Deform
曲線を変形、ソリッドおよびシートを変形するための統一コマンド。
Deform Solid and Sheet
不規則および規則的な表面にオブジェクトをラップします。
Extrude
E
面、領域、曲線を押し出します。
Fillet
B
曲線のフィレット、頂点のフィレット、シェルのフィレットのための統一コマンド。
Fillet Shell
B
エッジをフィレットまたは面取りします。
Hollow
ソリッドを空洞にします。
Loft
L
面、エッジ、曲線をロフトします。
Patch
シートの穴や閉じた曲線をシートに填埋します。
Pipe
P
エッジと曲線からパイプを作成します。
Raise Degree
Shift+S
曲線の次数を上げ、表面の次数を上げるための統一コマンド。
Raise Surface Degree
Shift+S
表面の制御点を増加させます。
Remove Fillets from Shell
ソリッドおよびシートのフィレットを削除します。
Revolve
領域または曲線を回転させます。
Sweep
Shift+P
領域または曲線をスイープします。
Square
明示的な制御により、度数とスパンを持つ表面を作成します。
Slide
Slide Curve CV、Slide Surface CVの統一コマンド。
Slide Surface CV
Shift+G
CVをサーフェス上でCVハルに沿ってスライドさせます。
Thicken
選択したシートに厚みを追加します。
Thicken Face
選択した面に厚みを追加します。
XNurbs
高品質で複雑な表面を作成します。
Sheet Commands
Bridge Surface
2つのシートをブリッジします。
Extend
曲線を拡張、シートを拡張するための統一コマンド。
Extend Sheet
シートの外側の1つ以上のエッジを拡張します。
Find Boundary Edges
シートの開口部のエッジを選択します。
Join
J
曲線を結合、シートを結合するための統一コマンド。
Join Sheets
J
一致するエッジを持つシートを結合します。
Reverse
カーブ反転とシート反転の統合コマンド。
Reverse Sheet
シートの法線の方向を逆転させます。
Unjoin
Alt+J
曲線を分割、面を分割、シェルを分割するための統一コマンド。
Unjoin Faces
Alt+J
ソリッドまたはシートから選択した面を分離します。
Unjoin Shells
Alt+J
選択したソリッドまたはシートからすべての面を個々の面に分離します。
Untrim
Alt+T
元の形状に基づいてシートを作成します。
Face Commands
Delete Face
Shift+Backspace
or
Shift+Delete
or
Shift+X
面を削除します。
Draft Face
面を傾斜させます。
Match Face
面を別の面に一致させます。
Offset Face
選択した面をオフセットします。
Rebuild
曲線を再構築、面を再構築するための統一コマンド。
Rebuild Face
面を再構築します。
Unwrap Face
面を展開します。
Edge Commands
Delete Redundant Topology
不要なエッジを削除します。
Extend Edge
エッジを拡張します。
Imprint
Shift+I
ボディボディをインプリント、曲線ボディをインプリントするための統一コマンド。
Imprint Body Body
Shift+I
ツールソリッドまたはシートと交差するターゲットソリッドまたはシートにエッジを作成します。
Imprint Curve Body
Shift+I
ツール曲線に投影されるターゲットソリッドまたはシートにエッジを作成します。
Isoparam
Ctrl+R
アイソパラメータを追加します。
Offset Curve
O
平面曲線のオフセット、領域のオフセット、面ループのオフセット、エッジのオフセットのための統一コマンド。
Offset Edge
O
選択したエッジをオフセットします。
Offset Face Loop
O
選択した面のアウトラインエッジをオフセットします。
