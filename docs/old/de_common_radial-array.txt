Radial Array
Tastenkürzel

Schaltfläche
Rechts von der Befehlsleiste
Duplizieren Sie Objekte in einem kreisförmigen Muster, indem Sie den Winkel und die Anzahl angeben. Radiale Duplizierung ist ebenfalls möglich.
Grundlegende Nutzung
Wählen Sie das Objekt zum Duplizieren aus.
Klicken Sie auf die Schaltfläche
rechts von der Befehlsleiste.
Klicken Sie auf den Punkt, den Sie als Zentrum festlegen möchten.
Die Duplikationsvorschau wird angezeigt.
Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
Befehlsdialog
Modus:
Gesamt:
Geben Sie die Gesamtdistanz an, bei der alle Duplikate gleichmäßig auf dem angegebenen Gradwert verteilt werden.
Abstand:
Geben Sie den Abstand für jedes der Duplikate an.
Degrees:
Geben Sie den Winkel für die Duplizierung an.
Number:
Geben Sie die Anzahl der Duplizierungen an.
Repeat:
Geben Sie die Anzahl der Wiederholungen in einem radialen Muster an.
Repeat distance:
Geben Sie den Abstand zwischen jeder Wiederholung in einem radialen Muster an.
Tastenkombinationen
Umschalt
+ Mausrad hoch
Add copy:
Erhöht die Anzahl der Duplizierungen.
Umschalt
+ Mausrad runter
Subtract copy:
Verringert die Anzahl der Duplizierungen.
Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
