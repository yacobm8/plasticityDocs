You can create a circle tangent to three curves.
Basic Usage
Click the
button on the toolbar.
Click on the first and second curve that the circle will be tangent to, optionally you can find the tangent to your third edge.
Move the cursor to adjust the size of the circle.
Command Dialog
Select first curve segment:
Click to select the first curve segment when it is highlighted.
Select second curve segment:
Click to select the second curve segment when it is highlighted.
Radius:
Specify the diameter.
Click OK in the command dialog or right-click to confirm.
The circle is created.
