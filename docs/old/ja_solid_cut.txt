Cut（カット）
コマンド
Cut
ショートカット
C
ボタン
推奨コマンドメニューで
ソリッドまたはシートをカットできます。
基本的な使い方
C
を押してコマンドを実行します。
ターゲットとカッターを指定します。
コマンドダイアログ
Select target bodies:
カット対象となるソリッドまたはシートを選択します。
対象を変更するには、
Select target bodies
をクリックしてハイライトされたら、対象を選択します。
Select cutters:
カッターとなるカーブまたはフェイスを選択します。
対象を変更するには、
Select cutters
をクリックしてハイライトされたら、対象を選択します。
キーボードショートカット
C
Cut Curves:
スケッチの
Cut Curve
コマンドに切り替えます。
Cut Curve
を参照してください。
S
Screen space:
カット方向を見た目の方向からにします。
コマンドダイアログのOKをクリックするか、右クリックで確定します。
