Curve Array（カーブに沿って複製する）
コマンド
Curve Array

-

複製数を指定して、オブジェクトをカーブに沿って複製します。回転、スケール、配置オプションが使用可能です。
基本的な使い方
複製したいオブジェクトを選択します。
コマンドパレットで
Curve Array
を実行します。
沿わせたいカーブを選択します。
コマンドダイアログとキーボードショートカットでオプションを設定します。
コマンドダイアログ
Number:
複製する数を指定します。
Twist:
カーブに沿って捻る角度を度数で指定します。
Scale:
カーブに沿って拡大縮小倍率を指定します。
Alignment
Parallel:
軌道に関わらず、平行に複製します。
Normal:
軌道に沿った角度で複製します。
キーボードショートカット
Shift
+ ホイール上回転
Add copy:
複製の数を増やします。
Shift
+ ホイール下回転
Subtract copy:
複製の数を減らします。
コマンドダイアログのOKをクリックするか、右クリックで確定します。
