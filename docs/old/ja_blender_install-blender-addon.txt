Blender Bridgeのセットアップ
Blender Bridgeの機能を使うには、Blenderへのアドオンのインストールと、Plasticityでの機能の有効化が必要です。
Blenderアドオンのインストール
公式のGitHubリポジトリからBlender Bridgeアドオンをダウンロードします。以下は最新バージョンへの直接のリンクです。:
plasticity-blender-addon-main.zip
(opens in a new tab)
Blenderを起動します。
通常の方法でplasticity-blender-addon-main.zipをBlenderアドオンとしてインストールします。
Blenderを開きます。
Edit > Preferences > Add-ons に移動します。
"Install..." をクリックし、ダウンロードしたzipファイルを選択します。
"[ ]Object: Plasticity" がリストに表示されるまでに時間がかかる場合があることに注意してください。
インストールが完了すると、3Dビューポートのサイドバーに"Plasticity" を見つけることができます。
Plasticityでの機能の有効化
Plasticityを起動します。
画面左上のPメニューからPreferencesを開きます。
左側のServerを開き、Enabledにチェックリストを入れます。
以上で、Blender Bridgeの機能を使う準備が整いました。
