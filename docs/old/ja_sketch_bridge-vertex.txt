Bridge Vertex（頂点をなめらかに繋げる）
コマンド
Bridge
ショートカット
-
ボタン

連続性を考慮してふたつの頂点を繋げることができます。
基本的な使い方
繋げたいふたつの頂点を選択します。
コマンドパレットで
Bridge
を実行します。
コマンドダイアログとキーボードショートカットでオプションを設定します。
コマンドダイアログ
Tension 1:
Value 1/2:
始点/終点の第1制御点(CV)から第3CVに影響するテンション値を指定します。
Tension 2:
Value 1/2:
始点/終点の第2CVと第3CVに影響するテンション値を指定します。
Tension 3:
Value 1/2:
始点/終点の第3CVに影響するテンション値を指定します。
Start curvature:
G0, G1, G2, G3:
始点での
曲線の連続性
を指定します。
End curvature:
G0, G1, G2, G3:
終点での
曲線の連続性
を指定します。
コマンドダイアログでOKを押すか、右クリックで確定します。
