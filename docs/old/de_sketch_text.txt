Erstellt Curves aus dem eingegebenen Text. Eingebaute Schriftarten und Größen können angegeben werden.
Grundlegende Nutzung
Verwenden Sie den
Text
Befehl in der Befehlspalette.
Legen Sie die Optionen im Command Dialog fest.
Befehlsdialog
Text box:
Geben Sie den Text ein, den Sie erstellen möchten.
Font:
Wählen Sie die gewünschte Schriftart aus.
Size:
Geben Sie die Schriftgröße an.
Bestätigen Sie mit OK im Command Dialog oder Rechtsklick.
Die Text Curves werden erstellt.
❗
Hinweis
Die Standard-Schriftgröße beträgt 1cm in der Höhe und der Text wird am Ursprung der XY-Ebene positioniert.
